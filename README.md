kxfw-example<br>
============<br>
这是kxfw工程的使用例子，是一个非常复杂的UI界面展示。<br>
而这界面原本是我以前创业开发的一个产品界面，透过这个界面，你应该能更好的掌握kxfw的使用，甚至更加深了Qt的了解。<br>
这个例子实现的功能演示包括如下：<br>
1.skins.xml的全局资源配置和非全布局资源配置的关系。<br>
2.泡泡提示，可随窗口拉伸而可自动跟随，有10多种跟随方式，只演示1种。<br>
3.KTextEdit控件的使用<br>
4.KMenu菜单的使用，共有10多种显示方式，只演示了一种，其它种只需改一个参数即可。<br>
5.MVC模式的List演示，tree控件演示可参考sxnote项目。<br>
6.自定义的复合式控件，kkslidervolume，这个类控件在实际中也比较常遇到。<br>
7.随内容变化而变化的ToolTip演示，显示方式也是有10多种。<br>
8.其它特性如窗口布局HBox,VBox,Anchor，Table四种。<br>
9.Resource/layout目录下放置了产品的所有UI布局文件，而DEMO只显示了主界面，其它文件可自由测试。<br>
<br>
演示效果，可到http://www.kxtry.com主页查看。<br>
